# 游戏配置

## emuera.config

`emuera.config` 是 Emuera 的配置文件，在游戏目录下。

启动时自动生成默认配置，可通过 Emuera 菜单修改。

### 主要配置项

| 项目 | 说明 | 默认值 |
|------|------|--------|
| フォント | 显示字体 | 系统默认字体 |
| フォントサイズ | 字体大小 | 14 |
| 前景色 | 默认前景色 | 白色 |
| 背景色 | 默认背景色 | 黑色 |
| 選択肢色 | 选项颜色 | 黄色 |
| 選択肢背景色 | 选项背景色 | 黑色 |
| フォーカス色 | 焦点颜色 | 红色 |
| メッセージスキップ | 跳过未读消息 | 无 |
| ウィンドウサイズ | 窗口大小 | 800x600 |
| セーブデータをバイナリ形式で保存する | 二进制存档 | 未选中 |

## _replace.csv

`_replace.csv` 放在 CSV 文件夹内，用于自定义游戏中的显示文字和行为。

### 可替换项目

| 项目 | 说明 |
|------|------|
| お金の単位 | 货币单位（默认 `$`） |
| DRAWLINE文字 | 分隔线字符（默认 `-`） |
| PALAMLVの初期値 | PALAMLV 初始值 |
| EXPLVの初期値 | EXPLV 初始值 |
| PBANDの初期値 | PBAND 初始值 |
| RELATIONの初期値 | 相性初始值 |
| 汚れの初期値 | STAIN 初始值 |
| 标题画面文字 | `[0] 从头开始` 等可替换 |

## config_files

### 配置项的强制

可以在游戏目录放置配置文件强制特定配置：

- `force_emuera.config`：强制配置

## _Rename.csv

文件名重定向文件。放在 CSV 文件夹内。

```
原名,1,新名,1
```

可将 CSV 和 ERB 中的关键字替换为其他文字（用于跨语言等场景）。

Emuera 默认不启用此功能，需在设置中勾选「使用_Rename.csv」。

## 调试模式

### 启用调试

通过 Emuera 菜单或命令行参数启用调试模式。

### 调试命令

| 命令 | 说明 |
|------|------|
| `DEBUGPRINT` | 调试输出 |
| `DEBUGPRINTV` | 调试数值输出 |
| `DEBUGPRINTS` | 调试字符串输出 |
| `DEBUGPRINTFORM` | 调试格式化输出 |

上述命令在非调试模式下被忽略。

### 调试变量

调试模式下可用以下特殊变量（非调试模式返回 0 或空字符串）：

| 变量 | 说明 |
|------|------|
| `DBG_CURRENTFUNC` | 当前函数名 |
| `DBG_CURRENTLINE` | 当前行号 |
| `DBG_CURRENTFILE` | 当前文件名 |

### `;#;` 调试专用行

```
;#;PRINTL DEBUG: MONEY={MONEY}
```

以 `;#;` 开头的行仅在调试模式下执行。

### 调试窗口

调试模式下可通过菜单打开调试窗口，查看：
- 变量值
- 函数调用栈
- ERB 源代码

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| F5 | 切换调试模式 |
| F9 | 断点设置/取消 |
| F10 | 单步执行 |
| F11 | 进入函数 |
| Shift+F11 | 跳出函数 |
