# 运算符

## 运算符优先级表

| 分类 | 优先级 | 可复合赋值 | 符号 |
|------|:------|:----------|------|
| 否定运算符 | 高 | × | `~`、`!` |
| 算術运算符 | ↑ | ○ | `*`、`/`、`%` |
| | | ○ | `+`、`-` |
| 位移运算符 | | ○ | `<<`、`>>` |
| 比较运算符 | | × | `<`、`>`、`<=`、`>=` |
| | | × | `==`、`!=` |
| 逻辑运算符 | | ○ | `&`、`|`、`^` |
| | ↓ | × | `&&`、`!&`、`||`、`!|`、`^^` |
| 三目运算符 | 低 | × | `?～#` |

## 单目运算符

| 符号 | 说明 | 优先级 |
|------|------|:------|
| `~` | 位取反（bitwise NOT） | 最高 |
| `!` | 逻辑否定（logical NOT） | 最高 |

```
A = ~B       ; 位取反
IF !FLAG:0  ; 逻辑否定
```

## 二目运算符

### 算术

| 符号 | 说明 | 例 |
|------|------|----|
| `+` | 加法 | `A + B` |
| `-` | 减法 | `A - B` |
| `*` | 乘法 | `A * B` |
| `/` | 除法（整数，向下取整） | `10 / 3 → 3` |
| `%` | 取模 | `10 % 3 → 1` |

```
MONEY = 15 + 34 + 25
MONEY = 150 + (100 - 50) * 10
MONEY = TIME * 10
MONEY = DAY % 7 * 10
```

`+` 和 `*` 也支持字符串运算：
- `+`：字符串连接（`"abc" + "def" → "abcdef"`）
- `*`：字符串重复（`"ab" * 3 → "ababab"`）
- 不能混合数值和字符串运算

### 复合赋值

| 符号 | 等价 |
|------|------|
| `+=` | `A = A + B` |
| `-=` | `A = A - B` |
| `*=` | `A = A * B` |
| `/=` | `A = A / B` |
| `%=` | `A = A % B` |
| `&=` | `A = A & B` |
| `|=` | `A = A | B` |
| `^=` | `A = A ^ B` |
| `<<=` | `A = A << B` |
| `>>=` | `A = A >> B` |

```
MONEY += 50        ; MONEY = MONEY + 50
MONEY *= 7 - 2     ; MONEY = MONEY * 5
MONEY %= TIME      ; MONEY = MONEY % TIME
```

### 位移

| 符号 | 说明 |
|------|------|
| `<<` | 左位移 |
| `>>` | 右位移 |

### 比较运算符

全部支持数值和字符串比较：

| 符号 | 说明 |
|------|------|
| `<` | 小于 |
| `>` | 大于 |
| `<=` | 小于等于 |
| `>=` | 大于等于 |
| `==` | 等于 |
| `!=` | 不等于 |

字符串比较：从头开始逐字比较，遇到不同字符时决定大小。

```
IF STR:0 == "字符串"
IF MONEY >= 1000
```

### 位运算符

| 符号 | 说明 |
|------|------|
| `&` | 位与（bitwise AND） |
| `|` | 位或（bitwise OR） |
| `^` | 位异或（bitwise XOR） |

### 逻辑运算符

| 符号 | 说明 |
|------|------|
| `&&` | 逻辑与（logical AND） |
| `||` | 逻辑或（logical OR） |
| `^^` | 逻辑异或（logical XOR） |
| `!&` | 逻辑与非（logical NAND） |
| `!|` | 逻辑或非（logical NOR） |

```
IF (A & 1p0) || (A & 1p3)    ; A 的第 0 位或第 3 位为 1
```

## 字符串赋值运算符

| 符号 | 说明 |
|------|------|
| `'=` | 使用字符串表达式进行赋值 |

```
STR '= "字符串"
STR '= TSTR:0 + "追加"
```

## 自增 / 自减

```
A++     ; A = A + 1
A--     ; A = A - 1
```

## 三目运算符

### 数值型

```
X = condition ? true_value # false_value
```

### 字符串型

```
STR '= \@ condition ? "true" # "false" \@
```

字符串型三目运算中 `#` 不可省略。

## 常量表示法

```
X = 32        ; 十进制
X = 0b100000  ; 二进制
X = 0x20      ; 十六进制
X = 1p5       ; 1 × 2^5 = 32
X = 13e3      ; 13 × 10^3 = 13000
```

注意：
- 不支持八进制（`012` 视为 `12`，不是 `10`）
- `p` / `e` 后只接受常量，不接受表达式
