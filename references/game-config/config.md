# 游戏配置

## emuera.config

`emuera.config` 是 Emuera 的配置文件，位于游戏目录下（与 Emuera.exe 同级）。

首次启动时自动生成默认配置，可通过 Emuera 菜单「帮助 → 设置」修改，也可以直接用文本编辑器编辑（下次启动生效）。

### 主要配置项

| 项目 | 说明 | 默认值 |
|------|------|--------|
| フォント | 显示字体 | 系统默认字体 |
| フォントサイズ | 字体大小 | 14 |
| 前景色 | 默认前景色 | 白色 |
| 背景色 | 默认背景色 | 黑色 |
| 選択肢色 | 选项颜色 | 黄色 |
| 選択肢背景色 | 选项背景色 | 黑色 |
| フォーカス色 | 焦点颜色 | 红色 |
| メッセージスキップ | 跳过未读消息 | 无 |
| ウィンドウサイズ | 窗口大小 | 800x600 |
| セーブデータをバイナリ形式で保存する | 二进制存档 | 未选中 |
| デバッグコマンドを使用する | 启用旧式 @ 调试命令 | NO |

> 调试模式相关功能（`;#;`、`DEBUGPRINT`、`[IF_DEBUG]` 等）详见 `references/reference/debug.md`。**不推荐在开发中使用。**
| オートセーブを行なう | BEGIN SHOP 时自动存档 | NO |
| セーブデータをsavフォルダ内に作成する | 存档放入 sav/ 子文件夹 | NO |

> **注意**：上表左侧是 `emuera.config` 中的实际键名（日文），不可改为中文。

### 配置文件优先级（三级覆盖系统）

Emuera 按以下顺序读取配置文件，**后读取的覆盖先读取的**：

```
csv/_default.config   →   emuera.config   →   csv/_fixed.config
```

| 文件 | 位置 | 作用 |
|------|------|------|
| `_default.config` | `csv/` 文件夹 | 仅当 `emuera.config` 不存在时生效，用于提供推荐的初始配置 |
| `emuera.config` | exe 同级目录 | 正常配置文件，用户可通过菜单修改 |
| `_fixed.config` | `csv/` 文件夹 | 强制配置，覆盖 `emuera.config`，**且对应项无法通过菜单修改** |

- **`_fixed.config`**：用于锁定对游戏运行至关重要的配置项（如行折返、背景色等）。只应设置必要的最小项，过度使用会剥夺用户的自定义权利。
- **`_default.config`**：用于推荐但非强制的初始值。用户可通过菜单覆盖。

> ⚠️ 文件名必须完全匹配（含下划线），放在子文件夹中或改名均无效。

## _replace.csv

`_replace.csv` 放在 `csv/` 文件夹内，用于自定义游戏中的显示文字和默认数值。

文件格式：每行一个项目，格式为 `项目名,值`。

### 完整可替换项目列表

| 项目 | 说明 | 默认值 |
|------|------|--------|
| お金の単位 | `PRINT_SHOPITEM` 显示价格的货币单位 | `$` |
| 単位の位置 | 货币单位位置：`前`（数字前）或 `後`（数字后） | `後` |
| 起動時簡略表示 | 关闭「加载时显示报告」后的替代显示文字 | `Now Loading...` |
| 販売アイテム数 | 购物时触发 `@EVENTBUY` 的商品编号范围（0～N-1） | — |
| DRAWLINE文字 | `DRAWLINE` 命令输出的分隔线字符 | `-` |
| BAR文字1 | `BAR`/`BARL` 进度条已填充部分的字符 | `*` |
| BAR文字2 | `BAR`/`BARL` 进度条未填充部分的字符 | `.` |
| システムメニュー0 | 标题画面选项 0 的文字 | `最初からはじめる` |
| システムメニュー1 | 标题画面选项 1 的文字 | `ロードしてはじめる` |
| COM_ABLE初期値 | 未定义 `@COM_ABLE{X}` 时的默认值（1=可执行，0=不可执行） | `1` |
| 汚れの初期値 | STAIN 初始化值，`/` 分隔 STAIN:0 / :1 / … | `0, 0, 2, 1, 8` |
| 時間切れ表示 | `TINPUT` 等限时输入命令超时时的显示文字 | `時間切れ` |
| EXPLVの初期値 | EXPLV 初始化值，`/` 分隔各元素 | `0, 1, 4, 20, 50, 200` |
| PALAMLVの初期値 | PALAMLV 初始化值，`/` 分隔各元素 | `0, 100, 500, 3000, 10000, 30000, 60000, 100000, 150000, 250000` |
| PBANDの初期値 | PBAND:0 的初始化值 | `4` |
| RELATIONの初期値 | `Chara**.csv` 中未指定时的相性默认值 | `0` |

> **注意**：上表左侧是 `_replace.csv` 中的实际键名（日文），必须原样使用。

## _Rename.csv

文件名重定向文件，放在 `csv/` 文件夹内。

```
原名,1,新名,1
```

可将 CSV 和 ERB 中的关键字替换为其他文字（用于跨语言等场景）。

Emuera 默认不启用此功能，需在设置中勾选「使用_Rename.csv」。
