# 快捷键参考

> ⚠️ **本文档内容为 Emuera 用户界面操作参考，不涉及代码编写。** 此处的快捷键由玩家在 Emuera 运行时直接使用，AI 生成的 ERB 代码无需也无法调用这些快捷键。

## 通用操作

| 快捷键 | 功能 | 条件 |
|--------|------|------|
| `Ctrl + B` | 最小化 Emuera 窗口 | 始终可用 |
| `Ctrl + C` / `Ctrl + Insert` | 打开剪贴板对话框（未选中文字时）；复制选中文字 | 始终可用 |
| `Ctrl + V` / `Shift + Insert` | 粘贴文字 | 始终可用 |
| `Esc` / 右键 | 跳过当前输出 / 中断宏执行 | 始终可用 |

## 调试相关（仅调试模式）

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + D` | 打开调试窗口 |
| `Ctrl + R` | 刷新调试窗口信息 |

## EM+EE 扩展

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + T` | 返回标题画面 |
| `Ctrl + R` | 重启 Emuera |
| `Ctrl + O` | 重新加载所有 ERB 文件（热重载） |

## 键盘宏

| 快捷键 | 功能 |
|--------|------|
| `F1` ～ `F12` | 调用已保存的键盘宏 |
| `Shift + F1` ～ `Shift + F12` | 将当前输入文字保存为键盘宏 |
| `Ctrl + 0` ～ `Ctrl + 9` | 切换宏分组（0～9） |
| `Esc` | 中断正在执行的宏 |

键盘宏也可通过 `macro.txt` 文件定义。此功能需要在 Emuera 配置中启用「キーボードマクロを使用する」。

> 注意：`ONEINPUT` / `ONEINPUTS` 命令执行期间键盘宏可能无法正常工作（引擎限制）。
